# Novaris Decision Center

Novaris is a decision support system designed for small business owners to track financial health and analyze operational risks. It provides a clear, advisor-like narrative on cash flow, revenue stability, and the cascading impact of project delays.

## Core Purpose

The system helps business owners answer critical questions:
- **"How much time do I actually have?"** (Cash Runway & Burn Rate)
- **"What happens if this project is late?"** (Delay Impact Simulation)
- **"Am I too dependent on one client?"** (Client Concentration Analysis)
- **"What should I do next?"** (Advisor-Driven Strategic Narratives)

## Key Features

### 1. The Decision Center
A unified dashboard that combines financial data with operational reality:
- **Strategic Verdict**: Human-centric narratives explaining current risks in plain English and Indonesian.
- **Risk Bridge**: Visualizes the connection between operational factors (like thin cash buffers) and financial outcomes.
- **Domino Chain**: Maps out how a single delay in delivery can cascade into a cash crisis.

### 2. Scenario Simulator
Test potential business decisions before making them:
- **Cash Injections**: See how new funding extends your survival time.
- **Overhead Reduction**: Calculate the immediate impact of cutting monthly bills.
- **Project Delays**: Adjust project timelines to see how much revenue is put at risk.

### 3. Integrated Risk Engines
- **Financial Engine**: Uses Monte Carlo simulations to estimate survival probability and cash resilience.
- **Operational Engine**: Analyzes project buffers, team sizes, and client dependencies to predict delay risks.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS (Vanilla CSS foundations).
- **Icons**: Lucide React.
- **Deployment**: Optimized for Vercel.

## Local Development

1. Install dependencies:
   ```bash
   bun install
   ```
2. Start the development server:
   ```bash
   bun dev
   ```
3. Build for production:
   ```bash
   bun build
   ```
